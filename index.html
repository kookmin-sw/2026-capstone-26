주요 기능

01 · 실시간 기록

이동 경로 기록

하루 동안의 위치 좌표가 지도 위 경로로 이어지고, 방문 장소와 하루 요약의 시작점이 됩니다.

위치 수집 ON/OFF 버튼
하루 기록 구성 요소
1. 방문 장소 리스트
2. 하루 메모
3. 위치 데이터 기반 하루 요약
02 · 다시 보기

날짜별
이동 경로 조회

과거의 하루를 다시 떠올리는 회고 기능으로, 날짜를 고르면 해당 날의 이동 경로를 불러옵니다.

날짜 선택
과거 날짜 조회
장소 리스트
03 · 자동 정리

체류 장소 자동 저장

사용자가 오래 머문 장소와 시간을 자동으로 저장해, 하루를 되짚는 기억의 단서로 남깁니다.
